Javascript

Javascript was developed in 1995. Brendan Eich was contracted by Netscape, a Microsoft competitor to create a language similar to Java, but to work server and client side.

Javascript surprisingly was developed in ten days!

Many parts of Javascript were hated and still are today, such as automatic semicolon insertion, lack of classes, unusual inheritance, and many other itmes.
